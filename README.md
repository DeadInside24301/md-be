Backend for my inventory system made from Python Flask and uses RESTFul Api's

Required to Install first:
Python 3.10 
pip any version (a new pip will be installed and used in venv)

Installation and Setup

Clone the repository using 'git clone', then navigate to the project directory using 'cd'.
Create the the venv by using this command 'python -m venv venv'
Activate venv by using 'source venv/Scripts/activate' (command may differ depending on the vs code terminal)(i use bash)
Install the requirements by using 'pip install -r requirements.txt'
Configure the interpreter by using ctrl + shift + p and type interpreter, selec python interpreter and 
              locate the python application within the projects venv folder (usually in C:\Users\[pcname]\[parent directory]\[Directory]\[project backend]\venv\Scripts\
Create a '.env' file in the root directory and configure the required environment variables.
Start the development server using 'python run.py'.
